## ⚙️ Cara Penggunaan

### 🔧 Langkah 1 — Persiapan
Pastikan kamu sudah memiliki executor Roblox yang terinstall di perangkatmu.
> Executor yang direkomendasikan: **Delta**, **Arceus X**, **Fluxus**, atau **Hydrogen**

---

### 🎮 Langkah 2 — Buka Game
Masuk ke Roblox dan buka game **Sambung Kata**.

---

### 📋 Langkah 3 — Copy Script

**Metode A — Loadstring:**
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/pearanoids/Sambung-Kata-Roblox/refs/heads/main/SBX_Script.lua"))()
```

**Metode B — Paste Langsung:**
> Buka file [`SBX_Script.lua`](./SBX_Script.lua) di repositori ini, copy seluruh isinya, lalu paste ke executor.

---

### ▶️ Langkah 4 — Jalankan
1. Buka executor kamu
2. Paste script di atas ke kotak script
3. Klik **Execute / Inject**
4. GUI akan muncul di layar
5. Klik tombol **ON** untuk mengaktifkan Auto Answer
6. Selamat bermain! 🎉

## 📚 Database Kosakata

Script ini menggunakan database kosakata Bahasa Indonesia yang telah dikurasi secara manual:

```
📊 Total kata  : ±3.000+ kata
🔤 Cakupan     : Huruf A hingga Z
📖 Jenis kata  : Kata dasar, kata turunan, frasa umum
🇮🇩 Bahasa      : Bahasa Indonesia
🔄 Diperbarui  : Berkala
```

Database tersimpan di Folder Kamus Besar Bahasa Indonesia dan bisa dilihat, diaudit, serta dikontribusikan secara bebas karena **open source**!
